位相空間におけるSober（ソバー）空間と空間の復元に関する詳細解説

本稿では、位相空間論および代数幾何学において極めて重要な概念である「sober空間」について、その定義と背景を述べた後、「開集合の束の情報から、元の空間の点集合と位相を完全に復元できるための必要十分条件が、空間がsoberであることである」という主定理の厳密な証明を行う。

1. Sober空間の定義と背景

位相空間 \(X\) が sober（ソバー） であるとは、以下の条件を満たすことを指す。

\(X\) の空でない任意の既約閉集合 \(C\) に対して、ある点 \(x \in X\) がただ一つ存在して、\(C = \overline{\{x\}}\) （点 \(x\) の閉包）となる。
  • 既約閉集合: 空ではなく、2つの真の部分閉集合の和集合として書けない閉集合。すなわち \(C = C_1 \cup C_2\)（\(C_1, C_2\) は閉集合）ならば、\(C = C_1\) または \(C = C_2\) が成り立つ。
  • 生成点 (generic point): その点の閉包が、考えている閉集合全体に一致するような点。

この概念は、Alexander Grothendieck らによる代数幾何学セミナー（SGA 4）において導入された。代数幾何学における \(\operatorname{Spec}(R)\) はザリスキー位相によりハウスドルフ性を欠くが、「各既約閉集合が唯一の生成点を持つ」というsober性を満たすため、幾何学的な空間としての妥当性を保っている。

2. 空間の復元定理の定式化

位相空間 \(X\) の開集合全体がなす完備分配束を \(\mathcal{O}(X)\) とする。ここから元の「点」を代数的に復元するため、以下の概念を導入する。

2.1 完全素フィルター (Completely Prime Filter)

\(\mathcal{O}(X)\) の部分集合 \(F\) が 完全素フィルター であるとは、以下の4条件を満たすことである。

  1. 全体集合の含有: \(X \in F\)
  2. 上方閉: \(U \in F\) かつ \(U \subseteq V\) ならば、\(V \in F\)
  3. 有限交叉性: \(U, V \in F\) ならば、\(U \cap V \in F\)
  4. 完全素性: 任意の開集合族 \(\{U_i\}_{i \in I}\) について、\(\bigcup_{i \in I} U_i \in F\) ならば、ある \(i \in I\) が存在して \(U_i \in F\) となる。
    ※注: \(I = \emptyset\) の場合を考えると、空集合 \(\emptyset \notin F\) であることもこの条件から導かれる。

2.2 復元された空間 \(\operatorname{pt}(\mathcal{O}(X))\)

\(\mathcal{O}(X)\) の完全素フィルター全体からなる集合を \(\operatorname{pt}(\mathcal{O}(X))\) とおく。ここに位相を導入するため、各 \(U \in \mathcal{O}(X)\) に対して以下の集合を定義する。

\[ \tilde{U} = \{ F \in \operatorname{pt}(\mathcal{O}(X)) \mid U \in F \} \]

\(\{\tilde{U} \mid U \in \mathcal{O}(X)\}\) は \(\operatorname{pt}(\mathcal{O}(X))\) の開集合系をなす。

このとき、各点 \(x \in X\) に対してその開近傍系 \(F_x = \{ U \in \mathcal{O}(X) \mid x \in U \}\) を対応させる自然な評価写像が定義できる。

\[ \eta_X : X \to \operatorname{pt}(\mathcal{O}(X)), \quad x \mapsto F_x \]

主定理: 評価写像 \(\eta_X\) が同相写像であるための必要十分条件は、\(X\) が sober 空間であることである。

3. 主定理の証明

証明は、幾何学的な対象である「既約閉集合」と、代数的な対象である「完全素フィルター」が1対1に対応するという重要な補題を経由して行う。

3.1 補題：完全素フィルターと既約閉集合の全単射

[補題] \(X\) の空でない既約閉集合の全体と、\(\mathcal{O}(X)\) の完全素フィルターの全体との間には、以下の対応により全単射が存在する。

  • \(C \mapsto F_C = \{ U \in \mathcal{O}(X) \mid U \cap C \neq \emptyset \}\)
  • \(F \mapsto C_F = X \setminus \bigcup_{U \notin F} U\)

[証明]
Step A: \(C\) が既約閉集合ならば \(F_C\) は完全素フィルターであること

  1. \(C \neq \emptyset\) より \(X \cap C \neq \emptyset\)。ゆえに \(X \in F_C\)。
  2. \(U \in F_C\) かつ \(U \subseteq V\) とする。\(U \cap C \neq \emptyset\) より \(V \cap C \neq \emptyset\) となり、\(V \in F_C\)。
  3. \(U, V \in F_C\) とする。もし \((U \cap V) \cap C = \emptyset\) だと仮定すると、
    \[ C = C \setminus (U \cap V) = (C \setminus U) \cup (C \setminus V) \]
    と書ける。\(C\) は閉集合なので、\(C \setminus U\) と \(C \setminus V\) は \(C\) の相対閉集合である。\(C\) の既約性より \(C = C \setminus U\) または \(C = C \setminus V\) となるが、これは \(U \cap C = \emptyset\) または \(V \cap C = \emptyset\) を意味し、\(U, V \in F_C\) に矛盾する。よって \(U \cap V \in F_C\)。
  4. \(\bigcup_{i \in I} U_i \in F_C\) とする。定義より \(C \cap \left( \bigcup_{i \in I} U_i \right) \neq \emptyset\)。分配則により \(\bigcup_{i \in I} (C \cap U_i) \neq \emptyset\) となるため、少なくとも1つの \(i \in I\) について \(C \cap U_i \neq \emptyset\)、すなわち \(U_i \in F_C\) が成り立つ。

Step B: \(F\) が完全素フィルターならば \(C_F\) は空でない既約閉集合であること

\(W_F = \bigcup_{U \notin F} U\) とおく（これは開集合である）。\(F\) の完全素性より、和集合 \(W_F\) 自身も \(F\) に属さない。したがって \(W_F \neq X\) であり、\(C_F = X \setminus W_F\) は空でない閉集合である。
\(C_F\) が既約であることを示す。\(C_F \subseteq A \cup B\) （\(A, B\) は閉集合）と仮定する。ド・モルガンの法則より、
\[ (X \setminus A) \cap (X \setminus B) \subseteq X \setminus C_F = W_F \]
\(W_F \notin F\) であり、\(F\) はフィルター（上方閉）なので、\((X \setminus A) \cap (X \setminus B) \notin F\) である。
もし \(X \setminus A \in F\) かつ \(X \setminus B \in F\) ならば、有限交叉性よりその共通部分も \(F\) に属してしまうため、少なくとも一方は \(F\) に属さない。
一般性を失わず \(X \setminus A \notin F\) とする。すると \(X \setminus A\) は \(W_F\) を構成する開集合の一つとなるため、\(X \setminus A \subseteq W_F\)。補集合をとって \(C_F \subseteq A\) となる。ゆえに \(C_F\) は既約である。

Step C: 互いに逆の操作であること

  • \(C_{F_C} = X \setminus \bigcup_{U \notin F_C} U = X \setminus \bigcup_{U \cap C = \emptyset} U\) である。\(\bigcup_{U \cap C = \emptyset} U\) は「\(C\) と交わらない最大の開集合」、すなわち \(X \setminus C\) に他ならない。ゆえに \(C_{F_C} = X \setminus (X \setminus C) = C\)。
  • \(F_{C_F} = \{ U \mid U \cap C_F \neq \emptyset \} = \{ U \mid U \not\subseteq W_F \}\)。\(W_F = \bigcup_{V \notin F} V\) であり、\(F\) は上方閉であるため、\(U \not\subseteq W_F \iff U \in F\) となる。ゆえに \(F_{C_F} = F\)。

以上で補題が示された。

3.2 評価写像 \(\eta_X\) の性質と主定理の完結

評価写像 \(\eta_X(x) = F_x\) は、点 \(x\) の閉包 \(\overline{\{x\}}\) に対応する完全素フィルターを与えていることに注意する（\(U \cap \overline{\{x\}} \neq \emptyset \iff x \in U\)）。

1. 単射性の同値条件（コルモゴロフ性）

\[ \eta_X(x) = \eta_X(y) \iff F_x = F_y \iff \overline{\{x\}} = \overline{\{y\}} \]

これが \(x = y\) を導くことは、任意の異なる2点が異なる閉包を持つこと、すなわち位相空間 \(X\) が \(T_0\)（コルモゴロフ）分離公理を満たすことと同値である。

2. 全射性の同値条件（生成点の存在）

\(\eta_X\) が全射であるとは、任意の完全素フィルター \(F\) に対し \(F = F_x\) となる \(x \in X\) が存在することである。補題より \(F\) はある既約閉集合 \(C\) と1対1に対応するため、これは「任意の既約閉集合 \(C\) に対して \(C = \overline{\{x\}}\) となる \(x \in X\) が存在する」ことと同値である。

3. 全単射性とSober性の同値性

上記1, 2より、\(\eta_X\) が全単射であることは、「任意の既約閉集合 \(C\) に対して \(C = \overline{\{x\}}\) となる \(x\) が一意に存在する」こと、すなわち \(X\) がsober空間であることと同値である。

4. 位相の同型性（同相写像であること）

\(\eta_X\) が全単射（\(X\) が sober）であると仮定する。\(\operatorname{pt}(\mathcal{O}(X))\) の基底開集合 \(\tilde{U}\) の逆像を計算する。

\[ \eta_X^{-1}(\tilde{U}) = \{ x \in X \mid \eta_X(x) \in \tilde{U} \} = \{ x \in X \mid U \in F_x \} = \{ x \in X \mid x \in U \} = U \]

逆像が開集合 \(U\) に一致するため \(\eta_X\) は連続である。同時に、\(\eta_X\) が全単射であるため順像 \(\eta_X(U) = \tilde{U}\) も開集合となり、開写像である。
ゆえに、\(\eta_X\) は同相写像である。（証明終）

4. 重要な具体例と非例（\(\operatorname{MaxSpec}\)）

4.1 Sober空間の例

  1. ハウスドルフ空間: \(\mathbb{R}^n\) や多様体など。強い分離性により、既約閉集合は単元集合 \(\{x\}\) に限られるため、自明にsoberの条件を満たす。
  2. \(\operatorname{Spec}(R)\): 代数幾何学の基本対象。Zariski位相において、閉集合 \(V(I)\) が既約であることと \(I\) の根基が素イデアル \(\mathfrak{p}\) になることが同値であり、\(\overline{\{\mathfrak{p}\}} = V(I)\) となる。
  3. Scott位相を持つ領域: 計算機科学において、有向完備半順序集合 (DCPO) 上のScott位相はsoberとなり、計算の極限値を幾何学的な「点」として厳密に扱える。

4.2 Soberではない空間の例：\(\operatorname{MaxSpec}(\mathbb{Z})\)

可換環 \(R\) の極大イデアル全体 \(\operatorname{MaxSpec}(R)\) は、一般にsoberではない。

  • \(R = \mathbb{Z}\) の場合、極大イデアルは素数 \(p\) が生成する \((p)\) のみである。
  • Zariski位相の制限により、この空間の閉集合は「有限集合」のみとなる（補有限位相）。
  • 空間全体 \(\operatorname{MaxSpec}(\mathbb{Z})\) は、2つの有限集合の和として書けないため既約閉集合である。
  • しかし、任意の点 \((p)\) の閉包は自分自身 \(\overline{\{(p)\}} = \{(p)\}\) であり、空間全体を生成することはできない。

\(\operatorname{Spec}(\mathbb{Z})\) に存在した零イデアル \((0)\) （\(\overline{\{(0)\}} = \operatorname{Spec}(\mathbb{Z})\) となる生成点）を意図的に取り除いてしまったため、代数的には存在するべき「全体を束ねる点」がトポロジーから欠落し、sober性が破壊されていることが分かる。
